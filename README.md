
# TechBridge Solutions Website

## Description
This project is a static website for **TechBridge Solutions**, a technology-focused company that provides media technology, IT solutions, and system integration services. The website presents the company’s mission, vision, core values, services, and contact information in a clean and responsive layout.

The goal of this project is to showcase how technology can bridge business needs with emerging digital solutions.


## Features
- Responsive navigation menu
- Hero section with smooth scrolling to services
- Company overview, mission, and vision
- Services section displayed using card layout
- Image gallery for installations
- Contact form with user feedback alert
- Simple and clean UI design


## Technologies Used
- HTML5  
- CSS3  
- JavaScript
